# Agentic Talent Manager

## Scraper Agent

MCP -> Scrapes jobs at X site for Y description
Questions:

1. I want you to get me all jobs that are related to DevOps on Hackernews that are remote

## Applying Agent
MCP -> Applies to a job listing using X information

1. I want you to apply to job listing http://megalargecompany.com/1/job using this resume as the information

## Core Agent
MCP -> Scrapes and applies for jobs
Tools are "scraper agent" and "applying agent"

## Tools

1. Tools do one job and one job well. MCP is a way to *expose* tools. Another "agent" CAN BE a tool as well.
